# 📊 Basic Aggregate Functions

This folder contains SQL problems that focus on **Aggregate Functions**, which are used to perform calculations on a set of rows and return a single value.

## 📚 Topics Covered

- `COUNT()`
- `SUM()`
- `AVG()`
- `MIN()`
- `MAX()`
- `ROUND()`
- Filtering aggregated results using `HAVING`
- Grouping data using `GROUP BY`

## 🧠 What I Learned

- How to calculate totals, averages, minimums, and maximums.
- The difference between `WHERE` and `HAVING`.
- How to group records based on one or more columns.
- How aggregate functions summarize data.
- How to combine `GROUP BY` with aggregate functions for data analysis.



## 💡 Key Takeaways

- Aggregate functions return a single value from multiple rows.
- `GROUP BY` creates groups before applying aggregate functions.
- `HAVING` filters groups after aggregation, whereas `WHERE` filters rows before aggregation.
- `ROUND()` is useful for formatting decimal results.
- Combining aggregate functions with joins helps solve real-world SQL problems.
